# CIWAGLINK VTU PLATFORM

CIWAGLINK is a production-ready Virtual Top-Up (VTU) platform built with Next.js, Supabase, and multi-provider redundancy.

Live URL: https://ciwaglink.vercel.app

---

## 🚀 FEATURES

- Airtime Top-up
- Data Subscription
- Cable TV (DStv, GOtv, Startimes)
- Electricity Bill Payment
- Exam Pins (WAEC, NECO, JAMB)
- Smart fallback system (CheapDataHub → iacafe)
- Supabase transaction storage
- Fully deployed on Vercel

---

## 🧱 STACK

- Next.js (App Router)
- Supabase (Database)
- Vercel (Hosting)
- CheapDataHub (Primary VTU API)
- iacafe (Backup VTU API)
- Vanilla CSS (No Tailwind)
